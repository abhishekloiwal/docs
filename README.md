# Mintlify Docs Draft

This folder is the public-facing Mintlify docs draft for hosted Atlas.

The intended audience is a customer or partner who is:

- using hosted Atlas
- connecting their own gateway
- exposing tools, workflows, approvals, and async jobs to Atlas

It is intentionally not the place for Atlas self-host or local product quickstart docs.

Run it locally with:

```bash
npx -p node@22 -p mint -c "mint dev --port 3002"
```

Open:

```text
http://localhost:3002
```

This draft is intentionally standalone so it can be moved into the landing/docs repository later without rewriting the content structure.
